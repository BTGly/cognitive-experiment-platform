## 遇到的问题

### 问题 1: HTTP2 协议错误
**现象**: `Error in the HTTP2 framing layer`
**解决**: 配置 Git 使用 HTTP/1.1
```bash
git config --global http.version HTTP/1.1
```

### 问题 2: 推送冲突
**现象**: `! [rejected] main -> main (fetch first)`
**解决**: 使用强制推送覆盖远程仓库
```bash
git push -u origin main --force
```

### 问题 3: Gitee Webhook 404 错误
**现象**: GitHub Webhook 调用 Gitee 返回 404
**解决**: 改用 Gitee 的"仓库镜像管理"功能，配置 Pull 自动同步
